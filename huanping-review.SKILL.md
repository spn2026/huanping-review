# 环评文件智能审核技能

这是一个技能文件，存放于仓库根目录（因环境限制）。功能与用户提示版本相同：

主要功能：
- 完整性检查（章节与必填项）
- 法规/标准映射与一致性检查（支持网络检索以补充法规条文）
- 数据表格与图表的完整性与单位校验
- 风险识别、高/中/低分级与缓解建议
- 输出结构化 JSON 报告、Markdown 摘要、以及带注释的 PDF

使用：在 Chat 中调用本技能并上传环评文档；技能支持数字 PDF、扫描件（需 OCR）和 DOCX 输入。
